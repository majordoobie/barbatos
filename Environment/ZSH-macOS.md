
# ZSH-macOS
- The [Kevin-Smets Repo](https://gist.github.com/kevin-smets/8568070) repo has all you need to install oh-my-zsh which is the package manager for zsh that we are using.

## Fixing paths
```
vim ~/.p10k.zsh
typeset -g POWERLEVEL9K_DIR_MAX_LENGTH=0
source ~/.p10k.zsh
```
